# Hi, I'm Celine 👋

I'm a Ph.D. researcher at TU Delft working on Spatial AI, Generative 3D, and XR systems.

My research focuses on building end-to-end research prototypes that combine VLM-based 3D scene understanding, generative 3D object creation, XR/MR interaction, and human-centered evaluation.

## Selected Projects

- **ElasticFit** — Fit-aware 3D object insertion via VLM reasoning and generative adaptation.
- **Skill-Adaptive Ghost Instructor** — A VR piano learning system with adaptive ghost-hand guidance and user-study evaluation.
- **UrbanSight** — A collaborative mixed-reality digital twin system for urban design workflows.
- **CityScope AR** — An AR system for visualizing real-time spatial data on a physical city model.

## Research Interests

Spatial AI · Generative 3D · XR Systems · Human-Centered AI · 3D Scene Understanding
